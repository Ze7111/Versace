# All Output Styling Options

### Color Types

* `red`
* `green`
* `yellow`
* `blue`
* `magenta`
* `cyan`
* `white`
* `black`
* `bold red`
* `bold green`
* `bold yellow`
* `bold blue`
* `bold magenta`
* `bold cyan`
* `bold white`
* `bold black`
* `bright_red`
* `bright_green`
* `bright_yellow`
* `bright_blue`
* `bright_magenta`
* `bright_cyan`
* `bright_white`
* `bright_black`

### Style Types

* `bold`
* `dim`
* `italic`
* `underline`
* `blink`
* `reverse`
* `hidden`
* `strike`

### Justify Types

* `left`
* `right`
* `center`
* `justify`

### End Types

| Type    | Description                                 |
| ------- | ------------------------------------------- |
| `newl`  | New line (2x Default)                       |
| `tab`   | Tab                                         |
| `rw`    | Return to the beginning of the line         |
| `space` | Single space                                |
| `endl`  | Keep the cursor on the end of the same line |
| `erase` | Erase the current line                      |
